
## 什么时候写 handoff

- 一次较长会话结束，且还有后续工作。
- 完成一个阶段性任务。
- 准备把任务交给另一个 AI 或未来的自己。
- `docs/work/current.md` 已经不足以承载继续工作所需细节。
