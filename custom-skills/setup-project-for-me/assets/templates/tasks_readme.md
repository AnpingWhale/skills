# 任务

这里记录可以被一个人或一个 AI 接走执行的工作单元。任务必须可读、范围清楚，并包含验收标准。

## 命名

```text
YYYY-MM-DD-task-slug.md
```

## 状态

- `待开始`
- `进行中`
- `已完成`
- `暂停`

## 任务与交付物

任务说明“为了推进项目或交付物，具体要做什么”。交付物本身登记在 `../../../deliverables/README.md`。
