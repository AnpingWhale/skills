# 知识库

这里沉淀项目中可复用的长期知识，包括背景信息、调研、踩坑、外部系统规则、领域概念和重要上下文。

## 放置规则

- 术语和缩写：更新 `glossary.md`。
- 单一主题知识：创建 `YYYY-MM-DD-slug.md`。
- 临时输入材料：不要直接堆在这里，先登记到 `inputs/README.md`，再把提炼后的知识放入本目录。
- 过时知识不要直接删除，先在文档中注明已过时或被哪个文件取代。
