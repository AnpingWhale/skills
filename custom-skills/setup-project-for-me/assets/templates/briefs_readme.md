# AI Briefs

这里存放给 AI 执行的 brief。默认面向单个 AI agent，但可以在 brief 中说明多 agent 分工。

## 命名

```text
YYYY-MM-DD-brief-slug.md
```

## 规则

- brief 应引用对应的 `docs/work/tasks/`、handoff 或决策文档。
- brief 不应成为长期事实源。
- 执行结束后，如果产生长期信息，应回写到 `docs/` 或 `deliverables/README.md`。
